## Tema

Automatización del proceso de solicitud de vacaciones mediante chatbot.

## Descripción

El proyecto consiste en la simulación de un chatbot capaz de gestionar solicitudes de vacaciones de empleados.

El sistema consulta una base de datos simulada, verifica la disponibilidad de días y aprueba o rechaza automáticamente la solicitud.

## Tecnologías utilizadas

* Python
* Excel
* BPMN 2.0
* GitHub
* Draw.io

## Estructura del proyecto

* chatbot.py → lógica del chatbot
* empleados.xlsx → base de datos simulada
* bpmn_vacaciones.png → diagrama BPMN
* informe_final.pdf → documentación completa
